How to install in Outlook:
1. Unzip this package.
2. Keep the HTML file and the folder named ms_fleet_signature_files in the same location.
3. Open MS_Fleet_Interactive_Outlook_Signature.html in Chrome or Edge.
4. Press Ctrl+A / Cmd+A, then Ctrl+C / Cmd+C.
5. Open Outlook > Settings > Mail > Compose and reply > Email signature.
6. Paste the signature and save.
7. Send a test email to yourself and click the phone/email/website areas.

Important:
- Do not paste only the PNG image if you want clickable links. The links come from the HTML.
- Outlook sometimes blocks remote/pasted images for recipients until they click “Download pictures.” That is normal for image-based signatures.
- For maximum reliability, host the signature image slices on your website and replace the local src paths with hosted URLs.

To edit links:
Open the HTML file in a text editor and change the href values:
href="https://msfleetservice.com"
href="mailto:Dispatch@msfleetservice.com"
href="tel:+18773498789"
